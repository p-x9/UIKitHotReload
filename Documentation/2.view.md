# view

| UIKit |
| ---- |
| UIView |

```json
{
  "class": "view",
  "id": "sample_view",
  "background_color": [1, 1, 1],
  "subviews":[
    {
      "class": "view",
      "id": "sub_view",
      "layout": {
      }
    }
  ]
}
```
