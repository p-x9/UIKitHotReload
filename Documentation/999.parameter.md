# border

```json
{
  border: {
    color:[0, 0, 0]
    width: 1
  }
}
```

|  UIKitHotReload | UIKit | 型 | description |
| ---- | ---- | ---- | ---- |
| color | layer.borderColor | [0.0~1.0, 0.0~1.0, 0.0~1.0, 0.0~1.0] | RGBAの順、 Aに指定がない場合は1.0になる |
| width | layer.borderWidth | Int |  |
